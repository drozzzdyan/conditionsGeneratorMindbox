Генератор условий mindbox

Возможны 3 основных случая:

1. Необходимо вывести 1 из n элементов, тогда сгенерируются условия для вывода только 1 блока, первого попавшего под условие

2. Если это значение n из n, тогда будут выводиться все доступные пользователю сегменты

3. Если же ввести k из n, где 1 < k < n, тогда выводятся первые k доступные сегментов пользователю, с учётом приоритета

Введите названия сегментов в поле ниже через ENTER
ВАЖНО: Вводите сегменты в порядке убывания их приоритета
